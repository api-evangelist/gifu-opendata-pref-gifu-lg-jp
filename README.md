# Gifu Prefecture Open Data (gifu-opendata-pref-gifu-lg-jp)

Gifu Prefecture Open Data is a regional government open-data portal for Japan running **CKAN**, exposing a standardized catalog API over ~1,881 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/gifu-opendata-pref-gifu-lg-jp/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=gifu-opendata-pref-gifu-lg-jp-api-evangelist&utm_content=repo)
